# Clashr-Auto-Desktop
### 无需学习，简单设置即可稳定使用！

## 软件截图
![Clashr Auto Desktop](https://raw.githubusercontent.com/ClashrAuto/Clashr-Auto-Desktop/master/uploads/logo.png "Clashr Auto Desktop")

## 支持协议
> ss,ssr,ssd,v2ray

## 支持平台

| 平台    | 全局（支持游戏） | 普通（只代理网页） |
| ------- | ---------------- | ---------- |
| windows | 支持             | 支持       |
| MacOS(m1)   | 支持             | 支持       |
| linux   | 支持             | 支持       |
