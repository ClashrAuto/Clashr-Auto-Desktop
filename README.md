> ss,ssr,ssd,v2ray

## 支持平台

| 平台    | 全局（支持游戏） | 普通（只代理网页） |
| ------- | ---------------- | ---------- |
| windows | 支持             | 支持       |
| MacOS   | 支持             | 不支持     |
| linux   | 不支持           | 不支持     |
